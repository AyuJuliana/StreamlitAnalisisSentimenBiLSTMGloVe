# Analisis Sentimen Bank Digital — Streamlit App

Aplikasi analisis sentimen ulasan Bank Jago menggunakan Custom BiLSTM + GloVe.

## Cara Menjalankan

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Download NLTK data (otomatis saat pertama kali jalan)

# 3. Jalankan aplikasi
streamlit run app.py
```

Buka browser di: http://localhost:8501
